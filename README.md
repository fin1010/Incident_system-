# Clinical / Safety Incident Recording System

This application supports the recording, review, and management sign-off of
clinical and safety incidents within care settings.

## Purpose
- Structured recording of incidents
- Management review and sign-off
- Audit integrity and inspection evidence
- Learning and service improvement

## Data Protection
- No incident data is stored in the code repository
- Data is stored in a secure database
- Access is role-based

## Deployment
- Application built with Streamlit
- Source code managed via GitHub
